# 格格足迹小程序

#### 介绍
格格足迹小程序1.0.0。

#### 预览地址
无

#### 软件架构
~~~
┌─biz                   一些业务
├─components            公共组件  
├─libs                  第三方组件
├─pages                 业务页面
├─static                静态资源
├─utils                 公共方法
│  └─api.js             接口地址文件     
└─app.json              全局配置类信息
~~~

#### 使用说明
~~~
# 安装依赖包
npm install

# 在小程序开发工具中构建
‘工具’-> ‘构建npm’

# 编译运行
在小程序开发者工具中运行
~~~
